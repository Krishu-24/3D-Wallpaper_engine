# 3D Wallpaper Engine

A desktop wallpaper engine that uses webcam-based head tracking to create dynamic perspective aware wallpapers.

The goal is to create the illusion of looking into a 3D scene by changing the displayed wallpaper based on the user's viewing position.

---

## Project Status

🚧 Research & Planning Phase

---

## Vision

Most wallpapers are static or animated.

This project aims to create wallpapers that react to the user's position, creating a convincing sense of depth and immersion.

By combining computer vision, graphics programming, and Blender generated assets, the wallpaper should behave like a window into a virtual environment.
